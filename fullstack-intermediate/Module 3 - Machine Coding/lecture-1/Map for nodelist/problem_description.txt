Problem Description:
Write a program in JavaScript to do the below mentioned task:

Map function does not exist for a nodelist like it does for an array. 
Create a function called nodeStyleMapper which takes a nodelist a css property name and a value and it add that property with the given value to all the dom nodes of the nodelist.
Using that function make the colour of all p tags red

Note:
The test case would check if all the p tags have color property set to 'red' or not.
