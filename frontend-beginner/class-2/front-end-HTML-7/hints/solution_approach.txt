Start by creating the basic structure of the HTML document using the <html>, <head>, and <body> tags.
Add ordered list using <ol> tag, and further add list items using <li> tag. Use "type" attribute set to "a" or "A" to change the list-item marker.
Add unordered list using <ul> tag, and further add list items using <li> tag. Use "type" attribute set to "square" to change the list-item marker.
